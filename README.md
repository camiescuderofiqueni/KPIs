# KPIs: Índices de Resiliencia Térmica

Librería de Python para el cálculo de índices de resiliencia térmica para zonas interiores. Incluye implementaciones para el Score de Resiliencia de Amaripadath, el Índice de Disconfort por Ocupación (IOD), Heat Index (HI), Wet Bulb Globe Temperature (WBGT), Horas de exceso (HE), Ambient Warmness Degree (AWD) y Overheating Escalation Factor (alpha_IOD).

## Instalación
```bash
pip install KPIs
